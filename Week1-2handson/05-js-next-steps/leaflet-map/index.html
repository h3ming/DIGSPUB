Interactive Map with Leaflet.js

This map uses the Leaflet.js library with tiles from OpenStreetMap.

Map Actions:

How it works:

  • Tile Layer: Map images loaded from OpenStreetMap servers
  • Markers: Interactive points with popups
  • Zoom: Use mouse wheel or +/- buttons
  • Pan: Click and drag to move around

Key Code:

// Create map centered on coordinates
const map = L.map('map').setView([lat, lng], zoom);

// Add tile layer
L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png').addTo(map);

// Add marker with popup
L.marker([lat, lng]).addTo(map).bindPopup('Hello!');
