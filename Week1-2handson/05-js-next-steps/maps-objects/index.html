JavaScript Maps and Objects

1. JavaScript Objects

Objects store key-value pairs as properties. Keys are always strings (or Symbols).

2. JavaScript Maps

Maps are designed specifically for key-value storage. Keys can be any type, and order is guaranteed.

Object vs Map: Quick Comparison

Feature Object Map
Key types Strings/Symbols only Any type
Key order Not guaranteed Insertion order
Get size Object.keys(obj).length map.size
Check key exists "key" in obj map.has(key)

When to use Object: For structured data with known properties (like a person or settings).

When to use Map: For collections where you'll add/remove items, or need non-string keys.
